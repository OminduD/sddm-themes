## 🌌 Overview

Hyprlands Video Themes is a collection of premium SDDM themes featuring looping video backgrounds, clean aesthetics, and flexible layouts. Designed with a focus on East-Asian minimalism and cinematic atmospheres, it transforms your login screen into an immersive experience.

## ✨ Features

- **🎬 Video Backgrounds:** High-quality, seamlessly looping video backgrounds.
- **🌸 Three Unique Variants:** 
  - 🏮 Ancient China Lantern
  - 🌸 Cherry Blossom
  - 🎋 Serene Bamboo Forest
- **🧑‍💻 User Profiles:** Beautiful user avatar support with fallback glowing icons.
- **🖥️ WM/DE Session Selector:** Easily switch between Hyprland, KDE Plasma, GNOME, etc.
- **⚙️ Dynamic QML Layouts:** Flexible UI components with support for different alignments (left, center, right), cool custom fonts, and a virtual keyboard toggle.
- **🔧 Interactive Installer:** An easy-to-use TUI script (`install.sh`) for seamless theme setup, dependency checking, and live previews.

## 🚀 Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/omindu/sddm-themes.git
   cd sddm-themes
   ```

2. Run the interactive installer script (requires `root` privileges to install to `/usr/share/sddm/themes/`):
   ```bash
   sudo ./install.sh
   ```

3. Follow the on-screen prompts to choose your variant, explore fonts, set alignments, and preview the theme.

## 🎨 Configuration

Each variant comes with its own configuration file found in the `Themes/` directory (e.g., `ancient_china_lantern.conf`). You can manually edit these to:
- Change the background video.
- Modify the theme fonts.
- Adjust form alignments and blur effects.
- Customize color accents.
